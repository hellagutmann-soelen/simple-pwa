Simple PWA

What is PWA?

A progressive web app (PWA) is a website that looks and behaves as if it is a mobile app. PWAs are built to take advantage of native mobile device features, without requiring the end user to visit an app store, make a purchase and download software locally. Instead, a PWA can be located with a search engine query and accessed immediately through a browser.

About this example

  • This is a super simple example how to integrate PWA for a simple webpage
  • It is actual a project which is hosted on Github and the website itself is being hosted Netlify.
  • Every time the code will be commited and uploaded the Website will be deployed automaticially

PWA: Not really easy to use for beginners

  • The PWA API is very huge and complex
  • you could technicially write your own Service Worker Code
  • Google created Workbox which makes it still hard to use
  • An example how to workbox can be found here(Fun fact: This project started to use VITE with its PWA plugin as well!)

How does it work?

  • This project uses Vite which is a super fast and easy to use frontend tooling kit.
  • There is a super easy to use Vite PWA Plugin to make the PWA experience a lot better
  • With vite you can test it locally and build it when you deploy.
  • This Github project is connected with Netlify, which means every time you upload your code the deploy process starts automaticially.

What did I do?

  • Created a github account and created a blank repository
  • Installed Git
  • Installed Nod
  • Clone this project with your terminal (git clone https://github.com/hellagutmann-soelen/simple-pwa)
  • Go into this folder (cd simple-pwa)
  • Ran npm create vite for some boilerplate code
  • Added the Vite PWA Plugin
  • Added a bit of custom text and code
  • Logged in to netlify with my github account and imported my github project
  • Done!
